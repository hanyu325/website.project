### 介紹:
###### 這是一個電腦版的網頁小遊戲，玩家需要控制貓咪來拯救主人。
###### 貓咪的主人是個陷入憂鬱症的女孩，貓咪要想辦法打斷主人自殺，並尋求其他人的幫助。

### 玩法:
###### 控制貓咪移動: wasd
###### 互動: q
